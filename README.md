# Escape The Catacombs is a game that I made in C++ to get used to some OOP and pointer concepts. Also, it won't be console-based (so it'll have an UI - 2D).

It will be a game where you are a "fighter/warrior/(maybe more categories added in the future)" and have to escape from some catacombs.

The structure of a game will be something like:
- catacomb 1 - Boss 1 - catacomb 2 - Boss 2 - catacomb 3 - Boss 3

The monsters you'll fight with are some mytical creatures. Difficulty increases level by level and new monsters can appear.
